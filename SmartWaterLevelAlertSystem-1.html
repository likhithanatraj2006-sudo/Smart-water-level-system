System Overview

Cross-section view of the Smart Water Level Alert System showing all components and their wired connections.

Ultrasonic Sensor

HC-SR04 mounted at top of tank. Emits 40kHz sound waves and measures echo time to calculate water distance.

🔌

Arduino UNO

Microcontroller that processes sensor data, calculates water level %, and triggers alert logic based on thresholds.

📶

GSM Module (SIM800L)

Cellular communication module connected to Arduino via UART. Sends SMS alerts when level is critical.

🔗

Wired Connections

Sensor → Digital pins 9,10 | Arduino → GSM via TX/RX pins | 5V/3.7V power supply lines.

The HC-SR04 ultrasonic sensor emits sound pulses that bounce off the water surface and return — measuring the distance to calculate water level.

HOW IT WORKS
The sensor sends a 10μs trigger pulse → emits 8 ultrasonic bursts at 40kHz → sound travels to water surface and reflects back → Arduino measures echo duration → Distance = (echo time × 0.034) / 2 → Level % calculated.

Arduino receives the echo signal, runs the distance formula, and computes the water level percentage in real time.

The system monitors three alert zones: Critical (<20%), Warning (20–40%), and Normal (>40%). Current reading of 16% triggers immediate SMS alert.

Arduino sends a serial command to the GSM module, which then dials into the cellular network to dispatch the SMS alert.

// Sending SMS via GSM AT Commands
void sendSMSAlert() {
  gsm.println("AT+CMGF=1");  // Text mode
  delay(1000);
  gsm.println("AT+CMGS=\"+917339370608\"");
  delay(1000);
  gsm.print("ALERT! Water level critically low.");
  gsm.write(26);  // Ctrl+Z to send
}█
AT COMMANDS SEQUENCE
Arduino sends AT+CMGF=1 (SMS text mode) → AT+CMGS (specify recipient number) → Message body → Ctrl+Z character (ASCII 26) to dispatch → GSM module acknowledges with "+CMGS: OK".

The GSM module successfully transmits the SMS alert to the registered mobile number. The user receives an immediate notification.
